+-------------+
| DESCRIPTION |
+-------------+

With this modded BBcode you can select all the quote by clicking on a span.


+--------------+
| INSTALLATION |
+--------------+

Upload / Overwrite / Join all files and folders to your ftp root.


+-------+
| NOTES |
+-------+

- Javascript come from phpBB3 forum functions and it was modified for PHP-Fusion
- $locale['bb_quote_select'] is set by default to "Select All" if not localized
- $locale['bb_quote_quote'] is set by default to "Quotation" if not localized
- There's a backup file called 'quote_bbcode_include.php.backup' with the original Wooya's BBcode
